Executive Summary
This plan establishes a legally compliant, three-phase structure for developing RareAngels from concept to
commercial blockchain platform while maintaining clear IP ownership, enabling non-profit prototype development,
and facilitating clean commercial spin-out.
Key Principle: Roderick retains 100% ownership of background IP and licenses it to RareStorm for charitable use.
RareStorm's role is as mission licensee and beneficiary, not IP owner. Commercial rights later transfer from Roderick's
holding entity to the Nevada trust structure at fair market value, with RareStorm receiving a perpetual charitable
license and revenue share.
Why This Works:
IRS Compliant: Non-profit uses IP for charitable mission; fair value paid for commercial rights
IP Protected: Patents/copyrights filed during prototype phase using overseer compensation
Clean Spin-Out: Proven model used by 23andMe, Flatiron Health, Color Genomics
Investor Ready: Clear IP chain of title, no ownership disputes
How This Differs from Sam Altman/OpenAI:
OpenAI Issue: Conflict between non-profit mission and for-profit subsidiary control
This Model: Clear IP ownership from day one; non-profit has limited license, commercial entity acquires full
rights at FMV
Key Difference: No dual-board structure or mission conflict; clean separation of charitable use vs.
commercial rights

Legal Foundation & Precedents
Why This Structure Is Legal and Common
1. IP Ownership by Inventor (Standard Practice)
Precedent: Individual inventors routinely own IP before licensing to entities
Legal Basis: U.S. Patent Law grants rights to "first to invent" or "first to file"
Compliance: No IRS issues when inventor licenses IP to non-profit for charitable use
2. Non-Profit IP Licensing (IRS Approved)
Precedent: Khan Academy, OpenMRS, Code for America all licensed IP from founders
IRS Rule: Non-profits can license IP for mission-related activities
Requirement: License must be at fair market value OR limited to charitable use
3. Commercial Spin-Out (Proven Model)
Precedent:
23andMe: Research project -> commercial company (founder-owned IP)
Flatiron Health: Academic research -> $1.9B acquisition (IP properly transferred)
Color Genomics: Foundation research -> VC-backed company
IRS Rule: Non-profit can sell/license IP to for-profit at fair market value

Detailed Step-by-Step Implementation Plan
Phase 1: IP Ownership Establishment (Month 1)

Actions:
1. Document IP Creation

2. File Provisional Patents

3. Copyright Registration
Register copyrights for code, documentation, algorithms
Cost: $65 per registration (estimate $500-1,000 total)
Protects source code and written materials
4. Trademark Applications
File "RareAngels" trademark (Class 42: Software services)
File "HopeCoin" trademark (Class 36: Financial services)
Cost: ~$350 per class ($700-1,400 total)
Deliverable: Complete IP portfolio owned by Roderick
Cost: $30,000 - $32,000
